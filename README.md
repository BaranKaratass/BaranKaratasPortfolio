# Baran Karatas - Portfolio Website

This repository contains the source code for my personal portfolio website. The site is designed to showcase my software development projects, technical skills, and professional journey.

## Technologies Used

* Frontend Framework: React 18
* Build Tool: Vite
* Language: TypeScript
* Styling: Vanilla CSS and CSS Modules
* Animations: Framer Motion
* Icons: Lucide React

## Project Structure

* src/components: Reusable UI components (Hero, ProjectCard, ProjectModal, etc.)
* src/data: Static data for projects, skills, and personal information
* src/styles: Global CSS rules and design tokens (variables)
* public/images: Project screenshots and other static assets

## Setup and Development

To run this project locally, follow these steps:

1. Clone the repository:
git clone https://github.com/BaranKaratass/BaranKaratasPortfolio.git

2. Navigate to the project directory:
cd BaranKaratasPortfolio

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

The application will be available at http://localhost:5173/

## Deployment

The project is configured to be easily deployed on platforms like Vercel, Netlify, or GitHub Pages. 

To build the project for production, run:
npm run build

The built files will be located in the dist/ directory.
